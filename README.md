# DaDaStore Static Website

A clean static marketing agency website for DaDaStore, built with plain HTML, CSS, and minimal vanilla JavaScript.

## Files

- `index.html` - semantic homepage markup with SEO and Open Graph metadata
- `assets/css/style.css` - responsive mobile-first styling
- `assets/js/main.js` - navigation, header state, current year, and lightweight reveal interactions

## Running Locally

Open `index.html` directly in a browser. No build tools, package installs, framework dependencies, or WordPress setup are required.

## Customization

- Update contact email links in `index.html`
- Adjust brand colors in `assets/css/style.css` under the `:root` variables
- Replace placeholder case studies in the Work section with real results when available
